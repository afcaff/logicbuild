# Logic Build for a course

This project was for a course taken in 2025. A 17 segment hex dispay that is controlled by an arduino nano using bit shifting. The project starts with assigned letters in this case 'MUHfp'. We then were given the challenge of converting from OR to NAND/NAND gates, followed by using K-maps to determine which segments should be turned on by each variable. The Logic Gates were then simulated in Logisim displaying our Logic in action. I decided to build a physical demonstration of this simulation, and that is this project.
